# spring-oauth2-oidc-client-integration
## Spring boot OAuth2 OIDC client integration

## Pre-requisites

* Developer Account with Github for OIDC setup
* Client Id
* Client Secret
* Application call back url ( ${base-url}/login/oauth2/code/${registration-id} )
